Example of TapToTop Plugin

Scroll down and you will see in the bottom right a button that you can tap in order to return to the top of the page.

To use this component, include gallery-taptotop in your YUI.use, and then plug it into a node, typically the body:

YUI().use('gallery-taptotop', function (Y) {

    Y.one('body').plug(Y.TapToTopPlugin);    

});
